## 🚀 **Sobre o Projeto**

O **ZPL Wizard** é uma aplicação web moderna desenvolvida para facilitar o trabalho com códigos ZPL (Zebra Programming Language). Com uma interface intuitiva e recursos avançados, permite visualizar, editar e gerar etiquetas de forma eficiente.

### ✨ **Funcionalidades Principais**

- 📝 **Editor de Código ZPL** - Syntax highlighting e autocomplete
- 👁️ **Visualização em Tempo Real** - Preview instantâneo das etiquetas
- 🎨 **Interface Moderna** - Design responsivo e intuitivo
- 💾 **Importar/Exportar** - Carregue arquivos ZPL ou salve suas criações

## 🚦 **Getting Started**

### **Pré-requisitos**

- Node.js (versão 18 ou superior)
- npm ou yarn

### **Instalação**

1. **Clone o repositório**

   ```bash
   git clone https://github.com/seu-usuario/zpl-wizard.git
   cd zpl-wizard
   ```

2. **Instale as dependências**

   ```bash
   npm install
   ```

3. **Execute em modo de desenvolvimento**

   ```bash
   npm run dev
   ```

4. **Acesse a aplicação**
   ```
   http://localhost:5173
   ```

### **Build para Produção**

```bash
npm run build
npm run preview
```

## 🎯 **Como Usar**

1. **Editar ZPL**: Cole ou digite seu código ZPL no editor
2. **Visualizar**: Veja o preview da etiqueta em tempo real
3. **Ajustar**: Modifique o código e observe as mudanças instantaneamente
4. **Exportar**: Salve seu código ZPL ou exporte como imagem

## 📄 **Sobre ZPL**

ZPL (Zebra Programming Language) é uma linguagem de programação para impressoras de etiquetas Zebra. É amplamente utilizada em:

- 🏭 **Indústria** - Etiquetas de produtos e inventário
- 🏥 **Saúde** - Identificação de pacientes e medicamentos
- 📦 **Logística** - Códigos de barras e tracking
- 🛒 **Varejo** - Etiquetas de preços e produtos
